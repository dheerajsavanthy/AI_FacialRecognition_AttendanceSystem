Built with A.I.

Argus Attendance is a sophisticated software that uses deep learning, computer vision and facial recognition technologies to identify students in the classroom and mark their attendance automatically. Facial Recognition is behind the ability to accurately recognize the individuals and ensuring their presence to be noted. Additionally, the data is stored and can be accessed anytime. You can also be notified by simply subscribing via e-mail.

openCV

openCV is open source python library which is very useful for real time computer vision applications. Using its capabilities, you can accomplish your tasks efficiently with this application!

Be Notified

Not only recognizing the individuals and marking the attendance, the application can be used to be notified of the same via email and keep track of it. Also unregistered attendees can be tracked and alert the person in-charge

Access Reports

The daily attendance sheets will be stored in a database. By accessing this database, you can generate the reports of all or any student when required.
